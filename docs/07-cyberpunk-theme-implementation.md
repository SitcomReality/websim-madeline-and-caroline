# Phase 7: Cyberpunk Inferno Theme Implementation

## Overview
Transform the entire visual experience to match the "Cyberpunk Inferno" and "Neon Flame Graffiti" aesthetic. This includes colors, fonts, animations, particle effects, and overall visual style. The theme should feel high-energy, edgy, and visually striking.

## Implementation Steps

### Step 1: Color Palette Definition
Establish a cohesive color palette based on cyberpunk aesthetics. Primary colors should be electric blues, hot pinks, neon greens, and flame oranges. Create a constants file with all theme colors defined. Include variations for different states (hover, active, disabled).

### Step 2: Typography Overhaul
Replace current fonts with cyberpunk-appropriate typefaces. Consider using fonts like Orbitron, Audiowide, or similar futuristic styles. Mix bold geometric fonts with glitchy effects for headers. Ensure readability while maintaining the aesthetic.

### Step 3: Neon Glow Effects
Add neon glow effects to UI elements, particles, and game objects. Implement glow using box-shadows, text-shadows, and potentially canvas glow effects. Make glows animated with pulsing or flickering. This should be performance-friendly.

### Step 4: Scanline and CRT Effects
Add optional scanline overlay effects and subtle screen distortion to give a retro-futuristic feel. Implement chromatic aberration on important UI elements. These effects should be subtle enough not to distract from gameplay.

### Step 5: Animated Backgrounds
Create dynamic animated backgrounds for menus and possibly the game. Use gradients, geometric shapes, and particle effects. Consider parallax effects for depth. Backgrounds should be visually interesting but not distracting.

### Step 6: UI Frame Design
Design custom borders and frames for UI panels using the neon flame graffiti concept. Create corner decorations, edge glows, and divider lines that feel cyber-punk. Add subtle animations to make UI feel alive.

### Step 7: Particle Theme Updates
Update all particle effects to match the theme. Fire should have more neon qualities, maybe with electric blue cores. Gasoline could have an iridescent sheen. Character effects should be vibrant and high-contrast.

### Step 8: Icon and Symbol Design
Create custom icons for tools, abilities, and status indicators. Use geometric shapes, circuit board patterns, and graffiti-inspired designs. Icons should be immediately recognizable and thematically consistent.

### Step 9: Animation Style Guide
Establish animation guidelines that match the theme. Use snappy, high-energy animations. Add easing functions that feel punchy. Include glitch effects and digital artifacts for transitions. Everything should feel kinetic.

### Step 10: Sound Design Preparation
While not implementing sound yet, document the type of audio that would match this aesthetic. Electronic music, synth effects, distorted impacts, and glitchy transitions. Structure code to easily add sound triggers later.

## Notes
The cyberpunk inferno theme should feel energetic and rebellious without being overwhelming. Every visual element should contribute to the atmosphere. Be careful not to sacrifice readability for style. Test all colors for accessibility and contrast. The theme should enhance the chaotic pyromania gameplay, not fight against it. Consider how the theme will look during intense gameplay moments versus calm menu moments.
